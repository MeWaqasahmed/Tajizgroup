# Tazij Meats - Professional B2B Website

A complete professional B2B website for Tazij Meats, Pakistan's top government-awarded halal meat exporter.

## Brand Identity

- **Company**: Tazij Meats (subsidiary of Tazij Group)
- **Tagline**: "Always Fresh"
- **Primary Color**: Forest Green #2D4A1E
- **Accent Color**: Gold #C8A84B
- **Background**: Cream #F7F2EA
- **Fonts**: Playfair Display (headings), DM Sans (body)

## Features

### Global Elements (on every page)
- Fixed navigation bar with logo and menu
- Floating WhatsApp button (bottom-right)
- Comprehensive footer with 4 columns

## How to Use

1. Open `index.html` in a web browser to view the website
2. All pages are linked through the navigation menu
3. The contact form redirects to WhatsApp for direct communication
4. Replace placeholder images in the `resource/` folder with actual company images

## Customization

### Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --forest-green: #2D4A1E;
    --gold: #C8A84B;
    --cream: #F7F2EA;
}
```

### Content
All content can be edited directly in the HTML files. Text is semantic and clearly structured.

### Images
Replace images in the `resource/` folder. Recommended sizes:
- Logo: 200x200px (transparent PNG)
- Hero images: 1920x1080px
- Team photos: 800x800px
